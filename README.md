Тестовое задание для AQA стажёра в компании учи.ру

## Структура проекта

```
|
├── tests/          — Дирректория с тестами         
│   ├── widget.page.ts      — Page Object: описание элементов и методов взаимодействия с виджетом
│   └── widget.spec.ts      — Автотесты: сценарии проверки функционала виджета
|
├── .gitignore              — Список игнорируемых файлов и каталогов
├── .nvmrc                  — Версия Node.js (22.14)
├── package-lock.json       — Фиксация версий зависимостей
├── package.json            — Описание проекта, зависимости и скрипты запуска
├── playwright-report/      — Сгенерированные отчеты о прохождении тестов (HTML)
├── playwright.config.ts    — Конфигурационный файл Playwright (URL, браузеры, отчеты)
└── README.md               — Документация проекта
```

Подготовка к работе
-------------------

Для начала работы убедитесь, что у вас установлены [node.js + npm](https://nodejs.org/en/download)

Установите зависимости проекта, выполнив команду:
```shell
npm ci
```




## Запуск тестов

### Все тесты сразу

```bash
npx playwright test
```

Запускает все тесты из директории `tests/` в браузере **Chromium**.

---

### Запуск каждого теста отдельно

#### Тест 1 — «Виджет поддержки открывается при клике на кнопку»

```bash
npx playwright test --grep "opens"
```

#### Тест 2 — «Форма обращения в поддержку имеет корректный заголовок»

```bash
npx playwright test --grep "has correct title"
```

#### Тест 3 — «Поле поиска в виджете принимает ввод и возвращает результаты» (дополнительный тест)

```bash
npx playwright test --grep "search returns results"
```

---

### Дополнительные режимы запуска

#### Запуск конкретного файла

```bash
npx playwright test tests/widget.spec.ts
```
